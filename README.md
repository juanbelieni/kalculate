# Kalculate

Kalculate is a simple math expression evaluator written in Rust.

## Usage

To use Kalculate, compile and run it with the expression you want to evaluate as an argument.

```bash
$ cargo build --release
$ ./target/release/kalculate "1 + 2 * 3"
1 + 2 * 3 = 7
```
